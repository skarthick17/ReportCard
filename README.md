# ReportCard

## Description
ReportCard is a Spring Boot web application for managing student report cards and marks.

## Features
- Student management
- Marks entry
- Teacher dashboard
- parent dashboard
- principal dashboard
- View report cards

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML/CSS
- Thymeleaf

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Configure MySQL database
4. Run ReportCardApplication.java
5. Open browser: http://localhost:8080
